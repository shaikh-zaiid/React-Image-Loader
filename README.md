# IMLoader

An application to upload your images to the cloud and get the link of the image to share it.

## Technology Stack

**Design**
- Figma

**Development**
- React
- React Context API
- Vitejs
- TailwindCSS
- Wouter

**Deployment**
- Vercel

## Getting Started
```
	# Install the required dependencies
	$ yarn install;

	# Run in development mode
	$ yarn dev;

	# Build app for production
	$ yarn build;
```
